## Physics Model

Classical machine (2nd-order swing equation) with constant voltage
behind transient reactance.  Network reduction via Kron elimination
of non-generator buses.

$$M_i \frac{d^2\delta_i}{dt^2} = P_{m,i} - P_{e,i} - D_i \frac{d\delta_i}{dt}$$

$$P_{e,i} = |E'_i|^2 G_{ii} + \sum_{j \neq i} |E'_i||E'_j|(B_{ij}\sin(\delta_i - \delta_j) + G_{ij}\cos(\delta_i - \delta_j))$$
