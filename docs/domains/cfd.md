# CFD Domain Pack

`pip install tensornet[cfd]`

## Modules

| Module | Description |
|--------|-------------|
| `tensornet.cfd.euler_1d` | 1D compressible Euler equations (shock tubes) |
| `tensornet.cfd.euler_2d` | 2D compressible Euler (oblique shocks, wedges) |
| `tensornet.cfd.godunov` | Godunov-type Riemann solvers |
| `tensornet.cfd.navier_stokes` | Viscous Navier-Stokes solver |
| `tensornet.cfd.les` | Large Eddy Simulation |
| `tensornet.cfd.hybrid_les` | Hybrid LES methods |
| `tensornet.cfd.multi_objective` | Multi-objective optimization |
| `tensornet.cfd.pure_qtt_ops` | Pure QTT CFD operators |
