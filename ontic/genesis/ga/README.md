## Overview

Geometric Algebra (GA), also known as Clifford Algebra, provides a unified mathematical framework for geometry, physics, and computer science. This module implements GA operations with QTT compression, enabling computations on exponentially large multivector spaces.

## Usage Examples

### Basic Multivector Operations

```python
from ontic.genesis.ga import CliffordAlgebra, vector, geometric_product

# Create 3D VGA
vga = CliffordAlgebra(p=3, q=0, r=0)

# Create vectors
a = vector(vga, [1, 0, 0])  # e1
b = vector(vga, [0, 1, 0])  # e2

# Geometric product
ab = geometric_product(a, b)  # e1e2 (bivector)

# Inner and outer products
from ontic.genesis.ga import inner_product, outer_product
dot = inner_product(a, b)    # 0 (orthogonal)
wedge = outer_product(a, b)  # e12
```

### Rotations with Rotors

```python
from ontic.genesis.ga import rotor_from_angle_plane, apply_rotor, bivector
import math

# Create rotation in xy-plane
B = bivector(vga, {(0, 1): 1.0})  # e12
R = rotor_from_angle_plane(math.pi/4, B)  # 45-degree rotor

# Rotate a vector
v = vector(vga, [1, 0, 0])
v_rotated = apply_rotor(R, v)
```

### Conformal Geometric Algebra

```python
from ontic.genesis.ga import (
    ConformalGA, point_to_cga, cga_line, cga_sphere, meet
)

cga = ConformalGA()

# Create geometric objects
p1 = point_to_cga(cga, [0, 0, 0])
p2 = point_to_cga(cga, [1, 0, 0])
line = cga_line(cga, p1, p2)

sphere = cga_sphere(cga, center=[0, 0, 0], radius=1.0)

# Intersection
intersection = meet(line, sphere)
```

### QTT-Compressed Multivectors

from ontic.genesis.ga import QTTMultivector, qtt_geometric_product
